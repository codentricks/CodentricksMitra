# CodentricksMitra
A simple tool to supercharge your Linux experience.

How to install: 

For Debian based distro like
==================================
Ubuntu, Linux Mint, Zorin OS ...<br>
Run command<br>
chmod 755 install_debian.sh
<br>
sudo ./install_debian.sh

For Fedora based distro like
==================================
Fedora, Nobara ...<br>
Run command<br>
chmod 755 install_fedora.sh
<br>
sudo ./install_fedora.sh

For Opensuse based distro
==================================
openSUSE Leap, openSUSE Tumbleweed,GeckoLinux  ...<br>
Run command<br>
chmod 755 install_opensuse.sh
<br>
sudo ./install_opensuse.sh

For Arch based distro
==================================
Garuda, CachyOs, Manjaro, ShaniOS ...<br>
Run command<br>
chmod 755 install_arch.sh
<br>
sudo ./install_arch.sh
<br>
For Flatpak support, follow url given below<br>
https://flathub.org/en/setup
